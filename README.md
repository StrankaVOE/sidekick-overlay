# Sidekick

**Game-aware overlay app for Riot Games (League of Legends, Teamfight Tactics, Valorant)**

---

⚠️ Alpha Version Notice

Sidekick is currently in **early alpha stage**.

* The **in-game overlay and champion select overlay are not yet uploaded/fully implemented**
* Current visuals represent a **design preview of the application and champion select UI**
* The purpose of this version is to demonstrate:

  * UI/UX direction
  * planned features
  * overall functionality

Despite being in alpha, the application is already designed with **full compliance in mind**.

🛡️ Compliance & Fair Use

Sidekick does **not provide any competitive advantage** and is built to comply with Riot Games policies:

* No gameplay automation
* No hidden or restricted information
* No real-time opponent tracking
* No unfair assistance during gameplay

All features are designed as **quality-of-life improvements only**.

---

🚀 Overview

Sidekick is a **lightweight, game-aware overlay app** that automatically detects which Riot game you are playing and adapts its interface accordingly.

The goal is to enhance the player experience through:

* contextual overlays
* useful tools
* clean, modern UI
* non-intrusive design

✨ Key Features

* Automatic game detection (LoL, TFT, Valorant)
* Context-based overlays per game
* Player profile insights
* **Riot-themed fun facts about your account**

  * account age
  * playstyle insights
  * unique stats and trivia
* Clean, rank-based visual design
* Modular feature toggling
